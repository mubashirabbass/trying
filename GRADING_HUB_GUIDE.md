# Grading Hub - Quick Start Guide

## 🎯 How to View Assignments (Row-Wise Display)

### Step 1: Access Grading Hub
```
Teacher Portal → Sidebar → Click "Grading Hub"
```

### Step 2: Select a Course (REQUIRED!)
```
You'll see a landing page with a course dropdown.
Select one of your courses from the list.
```

**Important**: You MUST select a course before you can see any assignments!

### Step 3: View Assignments
```
After selecting a course, you'll see:
- Statistics (Total, Pending, Graded)
- Search and filter options
- Assignment submissions in ROW-WISE CARDS
```

---

Each assignment submission appears as a full-width card:

```

│  👤 Student Name                          ✓ Graded / ⏳ Pending │

│                                                                │
│  💬 Feedback: "Great work on this assignment!"                │

```

---

## 🔍 Features in Each Card

### Left Side:
- **Student Avatar**: First letter of student name
- **Student Name**: Full name
- **Assignment Title**: Name of the assignment
- **Submission Date**: When it was submitted
- **Late Badge**: Shows if submitted after due date
- **File Link**: Click to view submitted file
- **Student Notes**: Any notes the student added

### Right Side:
- **Status Badge**: Graded (green) or Pending (amber)
- **Score Display**: Large number showing marks (if graded)
- **Grade Button**: "Grade Now" or "Edit Grade"

### Bottom (if graded):
- **Feedback Preview**: Shows your feedback to the student

---

## 🎨 Visual Indicators

### Status Colors:
- 🟢 **Green (Emerald)**: Graded
- 🟡 **Amber**: Pending Review
- 🔴 **Red**: Late Submission

### Badges:
- `✓ Graded` - Assignment has been graded
- `⏳ Pending Review` - Waiting for grading
- `🕐 LATE SUBMISSION` - Submitted after due date

## 🔧 How to Grade

### Step 1: Click "Grade Now"
```
Click the "Grade Now" button on any pending submission
```

### Step 2: Review Submission
```
Dialog opens showing:
- Student name
- Assignment title
- Submitted file (with preview for PDFs)
- Student notes
```

### Step 3: Enter Grade
```
- Enter marks in the "Marks Awarded" field
- Add feedback in the text area
- Click "Submit Grade"
```

### Step 4: Verify
```
The card updates to show:
- Green "Graded" badge
- Score displayed
- Feedback preview at bottom
```

## ❓ Troubleshooting

### "I don't see any assignments"

**Possible Reasons**:
1. ❌ You haven't selected a course yet
   - **Fix**: Select a course from the dropdown

2. ❌ No students have submitted assignments
   - **Fix**: Wait for students to submit, or check if assignments exist

3. ❌ You're not assigned to this course
   - **Fix**: Contact admin to assign you to the course

4. ❌ No assignments created for this course
   - **Fix**: Go to My Courses → Course Builder → Assignments Tab → Create

### "Assignments showing in table, not rows"

**Check**: Make sure you're on the latest version
- The page should show full-width cards, not a table
- Each submission should be a separate card
- If you see a table, the page didn't update

### "Can't select a course"

**Check**:
1. Are you logged in as a teacher?
2. Do you have any courses assigned?
3. Try refreshing the page
