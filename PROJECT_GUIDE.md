# Edu-Sphere: Modern LMS & Service Platform

## 🎯 Project Vision
Building a premium, enterprise-grade Service Website and Learning Management System (LMS) designed for the Pakistani and global markets. The focus is on high-fidelity design, professional service representation, and a seamless mobile-responsive experience.

## 🚀 Core Principles
1. **Premium Aesthetics**: Clean white and professional blue theme with modern UI components (Glassmorphism, Snap Scrolling, High-contrast typography).
2. **Mobile First**: Every section must maintain its integrity and usability on small screens.
3. **Step-by-Step Evolution**: We are building the platform module by module, starting with the public-facing landing experience.
4. **Deployment Ready**: Code is architected to be production-ready for deployment on modern cloud platforms.

## 📝 Important Instructions for Development
*   **Uniformity**: Always use the standard padding classes (`px-4 md:px-10 lg:px-16`) for new sections.
*   **Icons**: Use `Lucide-React` for all visual elements to maintain a consistent style.
*   **Responsiveness**: Always test new components on mobile widths (375px - 425px) before finalizing.
*   **Design Tokens**: Stick to the Emerald (Success/Action) and Professional Blue (Trust/Corporate) color palette.
