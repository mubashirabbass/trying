## 🔄 The Auto-Fill Process (Technical)

When you click "Select" on Ahmad Ali:

### **1. Frontend Fetches Data:**
```javascript
GET /api/users/123/details

Response:
{
  "id": 123,
  "fullName": "Ahmad Ali",
  "fatherName": "Muhammad Ali",
  "rollNumber": "GSC-01",
  "registrationNumber": "2026-GSC-01",
  "phoneNumber": "03001234567",
  "dateOfBirth": "2000-10-05",
  "profilePicture": "https://cloudinary.../photo.jpg",
  "enrollments": [
    {
      "enrollmentDate": "2026-01-15",
      "course": {
        "title": "Web Development",
        "duration": "6 months"
      }
    }
  ],
  "payments": [
    {
      "amount": 15000,
      "receiptNumber": "FEE-2026-001",
      "paymentDate": "2026-01-15"
    }
  ]
}
```

### **2. Frontend Auto-Fills Form:**
```javascript
useEffect(() => {
  if (student) {
    setFormData({
      rollNo: student.rollNumber,              // "GSC-01"
      admNo: student.registrationNumber,       // "2026-GSC-01"
      admDate: student.enrollments[0].date,    // "2026-01-15"
      traineeName: student.fullName,           // "Ahmad Ali"
      fatherName: student.fatherName,          // "Muhammad Ali"
      dob: student.dateOfBirth,                // "2000-10-05"
      mobile: student.phoneNumber,             // "03001234567"
      fromDate: student.enrollments[0].date,   // "2026-01-15"
      admFee: student.payments[0].amount,      // "15000"
      admFeeRecNo: student.payments[0].receiptNumber, // "FEE-2026-001"
      photo: student.profilePicture,           // "https://..."
      // ... all other fields
    });
  }
}, [student]);
```

### **3. Form Updates Instantly!**
```
All input fields now show the student's data ✓
Photo appears (if available) ✓
Total auto-calculates ✓
Ready for any manual edits ✓
```
